## Objective
Build a streamlined MVP of NovaSystem with dual interfaces (CLI + Browser App) that demonstrates parallel multi-agent problem-solving.

## Architecture Decisions
- **New Standalone Directory:** `nova-mvp/`
- **Backend:** Python + FastAPI (centralized API)
- **Frontend:** Svelte (lightweight, modern)
- **LLM:** Claude/OpenAI with swappable provider pattern
- **Agent Model:** DCE, CAE, Domain Experts running in parallel
