## Objective

The NovaSystem Modern UI already has an excellent foundation with:

  --primary-100: #dbeafe;
  --primary-500: #3b82f6;
  --primary-900: #1e3a8a;

  /* Animation Variables */
  --animation-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);
  --animation-smooth: cubic-bezier(0.4, 0, 0.2, 1);

  /* Component Variables */
  --button-height-sm: 32px;
